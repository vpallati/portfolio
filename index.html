SIMULATION OF SATELLITE PATH FROM EARTH TO THE MOON IN A 2D ENVIRONMENT DEVELOPED IN UNITY3D USING THE GENETIC ALGORITHM(2016)

I designed this project to test the genetic algorithm. I have designed a 2D simulation of the earth and the moon with rotation and revolution. A bunch of satellites will be launched at different points on the earth with different speeds and at different angles. This process is done for every generation. Genetic algorithm takes in properties from the previous launch data and yields new launches with best properties with a minute percent of mutation. https://github.com/vpallati/Genetic-Algorithm-MissionMoon

ULTIMATE TICTACTOE GAME (UNITY3D)

UltimateTicTacToe is a game designed in Unity3D game engine for which I have used an AlphaBeta pruning method to design its AI. The AI scans the board and scores each possible next move and selects the move with the best score.

NEURAL NETWORK TEST ON WINE CLASS PREDICTION USING PYTHON

I have developed this project to test the concepts of neural networks. I have taken a data set of different wines and their categories. 2/3 of the data was taken as training set and remaining as the test set and was tested with the trained neural system. The results were 95% accurate. https://github.com/vpallati/AI_NeuralNetworks
